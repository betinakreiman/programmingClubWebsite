Orbit App
A neat scroller app game to play when class gets boring. We used swift/xcode and even made the music with garage band. It's now relased in the app store!
North Parking Availability App
The Newton North Parking Availability App is a great way for students to find parking around the school. Most kids with their license are aware of the chaos and disorder there is around North when trying to find parking. The app is a great way to locate parking easily and quickly.
French Club Website
The French Club is another club at North that the Programming Club made a site for! We used Bootstrap and brilliant thinking to artistically craft a pleasing and visually appealing site that greatly benefits the French Club. Au revoir!
2048 Python
One of our club members was inspired by the classic 2048 game that so many people play during class. They recreated the game using python and user input. It's a fun and entertaining way to spend class and is really cool because he made it himself. Demo of game coming soon!
Old Club Website
After the pandemic, the club decided it wanted to entirely redo the website (taking ideas from the old website). We completely redid it trying to add the most useful information possible while still making it cool and modern.
Podtalk App
A poscast app for ios. They also had merch for sale to advertise for the club! Was published on app store and was a huge hit!
3D OOP Lua game
It's an action combat multiplayer fighting game. We used an object-oriented version of Lua and blender. The physics engine used was roblox studio. We took trigonometry we learned in class and applied it to the game's 3D space. The game is now released on the roblox platform. Over the course of the 9 month development period we hired various gfx artists and 3D modelers online to create art assets.
Website for TigerInsider
The Tiger Insider website uses the bootstrap CSS framework and silverstripe PHP for the backend. The club created some graphics to slap on the website and hired a gfx artist online to create an article placeholder banner. We like to use our own code, so bootstrap CSS was the optimal way in efficiency and creativity.
Dreamher Charity Website
The website made for Dreamher Charity was also created with bootstrap to create a better user-interface. This site was created for an organization. We hope to continue helping out charities near us by creating websites and games.
